# Contextos (Contexts)

Para gerenciamento de estado global e persistência de dados sensíveis, utilizamos a **React Context API**.

## Contextos Disponíveis

### `AuthContext.tsx`
*   **Função**: Gerencia o estado de autenticação do usuário.
*   **Dados**: Armazena o token JWT e informações básicas do perfil.
*   **Métodos**: Provê funções `login` e `logout`.
