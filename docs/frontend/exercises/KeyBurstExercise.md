# Key Burst Exercise

O **Key Burst Exercise** é um exercício de escrita rápida onde letras ou palavras "explodem" e o usuário deve digitá-las antes que desapareçam ou dentro de um tempo limite.

## Lógica de Jogo

- **Virtual Keyboard**: Inclui uma representação visual das teclas para auxiliar usuários iniciantes.
- **Vidas (Lives)**: O usuário tem um número limitado de erros permitidos antes do fim do exercício.
