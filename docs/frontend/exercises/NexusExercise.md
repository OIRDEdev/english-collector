# Nexus Exercise

O **Nexus Exercise** foca em conexões semânticas entre uma palavra central e várias opções periféricas.

## Visual e Interação

O exercício apresenta um "nódulo" central que representa o conceito principal. O usuário deve selecionar a opção que melhor se conecta a este nódulo.

## Estrutura Atômica

- `CentralNode.tsx`: Componente visual que exibe a palavra central com animações suaves de "pulsação".
- `NexusOption.tsx`: Botões estilizados para as escolhas do usuário.
- `useNexusLogic.ts`: Gerencia o estado da seleção única e a transição para o estado final após a escolha.
