# Connection Exercise

O **Connection Exercise** exige que o usuário associe pares de palavras (ex: sinônimos, antônimos ou tradução).

## Mecânica

O usuário clica em uma palavra de uma coluna e em sua correspondente em outra.

## Implementação

- **Shuffle Estável**: As palavras são embaralhadas apenas uma vez no início do exercício para evitar que mudem de lugar durante re-renders.
- **Keyboard Navigation**: Suporte completo para navegação via teclado, permitindo selecionar e confirmar pares sem o uso do mouse.
