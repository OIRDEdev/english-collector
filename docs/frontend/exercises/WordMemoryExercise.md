# Word Memory Exercise

O **Word Memory Exercise** é um exercício de alta velocidade focado em testar a memória imediata e a tradução rápida.

## Dinâmica do Jogo

Palavras aparecem rapidamente com sua tradução sugerida. O usuário deve decidir instantaneamente se a tradução está correta ou errada.

## Funcionalidades de Performance

- **Flashcard**: O componente de exibição da palavra é altamente otimizado para trocas rápidas de conteúdo.
- **Streak Logic**: O hook `useWordMemory` mantém um contador de acertos consecutivos (streak), que aumenta a pontuação final de forma exponencial.
- **Timer de Pressão**: Um cronômetro visual decrescente força a tomada de decisão rápida.
